# How to use

## Install
- python -m virtualenv venv
- source venv/bin/activate
- pip install -r requirements.txt
- pip install https://github.com/kivymd/KivyMD/archive/master.zip

## Env
- create .env file on root folder
- put SECRET_KEY="your-secret-key"

## Run
- python3 main.py
